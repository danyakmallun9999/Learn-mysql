## Setup installation

1. Install module

```javascript
    npm install
```

2. Ubah user & passowrd
```javascript
var db = mysql.createConnection({
    host: "localhost",
    user: "root",
    password: "daN00b-9999",
    database: "petani_kopi"
});
```
